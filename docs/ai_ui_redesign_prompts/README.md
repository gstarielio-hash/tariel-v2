# Prompts de Redesign por Área

Estes arquivos servem para você colar em outra IA e pedir uma nova proposta visual para cada área do web.

Objetivo:
- preservar a função e a hierarquia operacional do produto
- ignorar a estilização atual
- deixar a IA propor o visual do zero
- dar contexto suficiente sobre a estrutura atual da área para a IA não trabalhar no escuro

Como usar:
- escolha um arquivo desta pasta
- cole o conteúdo inteiro na IA de design/UI
- se quiser, anexe screenshots da tela atual só como referência funcional, não estética
- se a IA aceitar imagens, mande 1 ou 2 screenshots da área junto com o prompt
- se a IA aceitar contexto extra, diga explicitamente: "use a referência atual apenas para estrutura e função, não para copiar o visual"

Áreas cobertas:
- `00_prompt_generico_esqueleto_projeto.md`
- `01_admin_geral.md`
- `02_admin_cliente.md`
- `03_portal_inspetor_chat.md`
- `04_mesa_avaliadora.md`
- `05_biblioteca_templates.md`

Regra geral para todos os prompts:
- manter o escopo funcional do sistema
- não copiar cores, componentes, grid, tipografia ou acabamento da interface atual
- criar nova direção visual com base na operação real da área
- preservar os blocos essenciais que o prompt lista como referência atual
