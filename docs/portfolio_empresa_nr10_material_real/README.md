# Portfolio Empresa NR10: Material Real

Workspace canonica para refino de familias NR10 a partir de material real ou baseline sintetica externa.

## Regra

- cada familia deve ter manifesto de coleta, briefing de refino, area de material bruto e area de pacote de referencia;
- os prompts operacionais devem permanecer genericos, sem ancoragem em empresa especifica, salvo instrucao explicita;
- a adaptacao final de linguagem e layout deve acontecer so depois da consolidacao do pacote de referencia da familia.
