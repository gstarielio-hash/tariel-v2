# Mobile Agent Ideas Backlog

Status: aberto para propostas do agente
Data de referência: 2026-04-14

## Objetivo

Registrar ideias novas de UX, produto ou operação que surjam durante a execução do agente, sem alterar automaticamente o comportamento do Tariel além do que já foi aprovado.

## Regras

- ideias aqui registradas não são automaticamente aprovadas;
- mudanças estruturais ou conceituais novas devem ser registradas primeiro aqui;
- cada ideia deve trazer motivo, impacto esperado e risco;
- melhorias superficiais de UX podem ser implementadas sem registro prévio quando estiverem claramente dentro do contrato já aprovado.
