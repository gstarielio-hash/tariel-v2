# Prompts de Logos dos Logins para Colar no ChatGPT

Use sempre esta imagem como referência visual principal:

- [`android-chrome-512x512.png`](/home/gabriel/Área%20de%20trabalho/TARIEL/Tariel%20Control%20Consolidado/web/static/img/android-chrome-512x512.png)

## Direção recomendada

O caminho mais bonito e mais consistente para os logins não é criar 4 símbolos diferentes.

O melhor é:

- manter o mesmo símbolo Tariel
- manter os mesmos traços, geometria e proporção do ícone atual
- variar só o nome horizontal e o label sutil do ambiente

Assim os 4 logins viram uma família premium e não 4 marcas soltas.

## Nomes recomendados

- Portal do Inspetor: `Tariel Inspector`
- Admin-Cliente: `Tariel Tenant Admin`
- Mesa Avaliadora: `Tariel Review Desk`
- Admin-CEO: `Tariel Command`

## Prompt mestre

Cole este prompt junto com a imagem de referência:

```text
Use the attached image as a strict visual reference.

I do not want a new random logo style. I want a logo family derived from this exact Tariel symbol.

Preserve the same symbol DNA, same monoline geometry, same angular construction, same centered balance, same stroke thickness, same dark navy base, and the same cold light line color. The result must feel like the same brand system, not a redesign from scratch.

Create 4 horizontal login logo lockups for the same platform family:

1. Tariel Inspector
   subtle label: Portal do Inspetor

2. Tariel Tenant Admin
   subtle label: Admin-Cliente

3. Tariel Review Desk
   subtle label: Mesa Avaliadora

4. Tariel Command
   subtle label: Admin-CEO

Rules:
- keep the symbol almost identical to the reference
- keep the symbol on the left and the wordmark on the right
- add the subtle environment label as a small secondary line or restrained capsule
- industrial, technical, premium, sober, geometric
- transparent background
- no gradients
- no 3D
- no glow
- no mascot
- no extra icons
- no startup look
- no decorative shapes

Keep these visual constants:
- deep navy close to #141b31
- pale icy stroke close to #d7dfef
- precise geometric linework
- shield / validation / technical control feeling
- strong readability on login screens

Improve only what helps polish:
- spacing
- wordmark refinement
- alignment
- subtle label elegance

Deliver:
- 4 horizontal logo lockups on transparent background
- 4 dark background previews
- 4 light background previews
- 4 symbol-only versions

Do not invent another brand language. Evolve the existing Tariel mark into a refined login logo family.
```

## Melhor ajuste estético

Se o ChatGPT gerar algo genérico, acrescente isto no fim:

```text
Make it feel like a serious industrial software platform used by engineers, operators, reviewers, and technical administrators. Prioritize structure, discipline, precision, and quiet confidence over marketing style.
```

## Observação prática

Se quiser máxima consistência visual:

- gere primeiro o símbolo e o lockup mestre
- depois peça as 4 versões trocando só o nome e o label

Isso reduz a chance de o ChatGPT te entregar 4 logos desconectadas entre si.
